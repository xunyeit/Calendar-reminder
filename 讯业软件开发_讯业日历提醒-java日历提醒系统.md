### 一、产品概述
**“****日历提醒****”是一款专为现代生活与工作场景设计的****智能时间管理工具****，旨在帮助用户高效规划个人事务、重要纪念日及团队协作日程。该应用以简洁直观的日历视图为核心交互界面，融合了****个人日程创建、农历公历双轨显示、团队共享协同****等核心能力，适用于职场人士、家庭用户、项目管理者以及需要长期记忆关键节点（如生日、祭日）的人群。**

**平台不仅支持日常事项的记录与提醒，更特别强化了对****传统节日、农历日期、逝者纪念日****等文化性时间节点的支持，兼顾实用性与人文关怀。通过手机端一键操作，用户可轻松实现“记事—提醒—共享—回顾”的全流程闭环管理，提升个人效率与组织协同水平。**

**“日历提醒”不仅是时间管理助手，更是****连接过去、现在与未来的数字日记本****，让每一个重要的日子都不被遗忘。**

### 二、核心功能
#### 1. **创建日程**
+ **用户可在任意日期点击“+ 创建日程”按钮，进入日程编辑页面。**
+ **支持输入标题（如“张三生日”“公司例会”）、选择具体日期和时间。**
+ **独特支持****农历/公历双选****：在“生日”或“逝世日期”字段中，可自由切换“公历”与“农历”模式，自动匹配对应年份与节气。**
+ **示例：用户可设置“2025年11月13日（乙巳年九月廿四）”作为某位亲人的祭日，并永久保存，系统将在每年同一天自动提醒。**

#### 2. **我的创建**
+ **查看由本人添加的所有日程记录，按时间顺序排列。**
+ **支持查看、编辑、删除、分享日程内容。**
+ **可查看历史事件，便于回忆与复盘（如“去年今天发生了什么”）。**
+ **每条日程均标注农历信息，增强传统文化感知。**

#### 3. **我的团队**
+ **支持创建或加入团队日历，实现多人协同管理。**
+ **团队成员可共同查看、添加、修改公共日程（如家庭聚会、项目会议、团建活动）。**
+ **提供“+ 创建团队”入口，方便用户组建家庭组、工作小组或亲友圈。**
+ **未来可扩展权限管理功能（如管理员、普通成员），确保数据安全与协作有序。**

### 三、产品价值
#### **用户价值：让时间更有意义，让记忆永不褪色**
+ **精准记住重要日子****：无论是亲人诞辰、还是逝者祭日，都能通过农历+公历双轨记录，确保万无一失。**
+ **情感关怀功能强****：特别适合用于家庭成员间的亲情维系，尤其是对长辈、故人的情感寄托。**
+ **提升生活秩序感****：从琐碎事务到重大节点，全部纳入日历统一管理，让生活更有节奏、更可控。**
+ **跨代际友好：既满足年轻人数字化习惯，也照顾老年人对农历文化的依赖，实现全龄覆盖。**

## 架构特点
### 架构特性
+ **前后端分离架构，独立开发，符合主流开发模式**
+ **前端以Vue3+Vite为主技术，AntdV为UI界面框架**
+ **后端SpringBoot3为基础，MybatisPlus为数据操作框架，Redis为缓存框架**
+ **Maven多模块管理，插件化开发，方便安装、卸载、升级，降低耦合**
+ **业务模块与API抽离，模块之间便捷引用**
+ **数据库设计精巧，字段规范、易于扩展**
+ **支持国产密码算法加解密，等保测评国产项目无压力**
+ **支持MYSQL、ORACLE、SQLSERVER、PGSQL等主流标准结构式数据库**
+ **支持达梦、人大金仓、南大通用、九有、瀚高、虚谷数据库等国产数据库**
+ **支持中创、宝蓝德、东方通等中间件**
+ **支持Windows、Linux操作系统、国产操作系统部署**

### 功能特性
+ **完善的系统基础功能，满足使用需求，避免重复造轮子**
+ **支持本地文件、阿里云文件、腾讯云文件、MINIO文件上传**
+ **支持本地邮件、阿里云邮件、腾讯云邮件发送**
+ **支持阿里云短信、腾讯云短信发送**
+ **B、C端双账号认证体系，会话治理各自独立**
+ **完善的登录日志、操作日志、异常日志等审计功能**
+ **完善的会话监控、数据源监控、系统监控等必备监控功能**
+ **支持组织机构、权限管理、定时任务、系统配置等基础功能**

### 安全特性
+ **采用SaToken轻量级 Java 权限认证框架，功能强大、学习成本低**
+ **支持登录认证、权限认证、单点登录、三方登录、OAuth2.0等认证模式**
+ **增强的RBAC权限设计，资源于接口独立授权，更加灵活**
+ **支持按钮级别细粒度独立授权，界面按钮动态展示**
+ **支持API接口注解式、路由拦截式鉴权，防止越界访问**
+ **独创的数据范围机制，每个接口都可以配置不同数据范围**
+ **支持限流防抖，防重复提交，有效阻止脏数据产生**
+ **密码、手机、身份证号等使用国密算法加密传输、加密存储，数据更安全**
+ **操作日志使用SM2进行完整性保护，满足安全审计要求**

### 界面特性
+ **Vue3 + Vite为基础，AntdV为界面UI框架，视觉风格清新简洁**
+ **精细化设计，注重界面的每一处细节，操作轻松友好**
+ **暗黑风格、经典菜单、双排菜单、多页签、目录坞、主题切换等功能应有尽有**
+ **统一的网络框架、API接口拦截框架，拿来即可上手**

+ **Node.js 14x**
+ **Java1.8**
+ **Mysql 5.7**
+ **Redis 6.2.8**
+ **阿里云 OSS – 可更换其它。**
